Space Tower Defense Download


Space tower defense was a prototype game developed in Unity by a few friends and I. Please note that this 'game' is a prototype, and only features some of the basic mechanics of a tower defense game. It also lacks balance, polish, and many other features that would be expected in a full game.

Things to note beforehand

  • When downloading from google drive, it may notify you that the file is too big to scan for viruses. If so, just press 'continue downloading'.
  • The menus are very bright blue, so it might hurt your eyes.
  • The best map is 'Hayden' since the other maps may not be updated with the later systems implemented.
  • When placing down a tower, you can press 'Esc' to stop placing.
  • There is not a button to close the game, so you will need to press 'Alt + F4' for windows or 'Cmd + Q' for mac or some other keyboard shortcut to close the game.

Download instructions

  • Click here to go to the download page.
  • Press the download button on the top left of the new page.
  • Once downloaded, extract the zip file.
  • Run First.exe in the extracted file.
  • Once you press play, I would recommend selecting "Hayden" in the map selection.
  • Enjoy!
